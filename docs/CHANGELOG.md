# CHANGELOG

NEON NEKO RUNNER のバージョン履歴です。

## v1.0 正式リリース

* README にプレイ環境・GitHub Pages公開手順を追記
* キャンバスをレスポンシブ対応（`max-width: 100%` / `height: auto`）に調整
* `index.html` に `description` メタタグを追加
* ヒットストップ・演出タイマーの優先順位に関するコメントを整理し、正式リリース版として確定

## v0.95 公開準備版

* README を拡充し、操作方法・主な機能・公開URL（GitHub Pages）を記載
* ダブルジャンプを実装（最大2回ジャンプ、専用パーティクル・リング・本体グロー演出を追加）
* ジャンプ・ゲーム開始処理を `handlePrimaryAction` に整理

## v0.9 ゲームフィール強化

* ヒットストップ（魚取得・レア魚・レベルアップ・実績解除時に1フレーム全体を停止）を追加
* 画面シェイク（レア魚・レベルアップ・ゲームオーバー時）を追加
* スコアのポップ演出・ハイスコア更新時の発光演出を追加
* 魚取得時のリングエフェクトをコンボ演出と統合し、コンボ段階に応じた色・発光を調整

## v0.8 魚収集・実績

* 通常魚・レア魚の出現とスコアボーナス（+200 / +500）を追加
* 魚取得時のスパークルパーティクル・ポップアップ表示を追加
* 実績システム（First Jump / Combo Master / Fish Hunter / Rare Collector / Cyber Legend の5種）を追加
* 累計魚取得数・レア魚取得数・解除済み実績を `localStorage` に保存
* 実績解除通知・実績一覧画面を追加

## v0.7 障害物バリエーション

* 障害物を6種類（block / yarn / bone / vacuum / dog / trashcan）に拡張し、種類ごとの形状を描画
* レベルに応じた障害物の出現解放（レベル3でyarn/bone/vacuum追加、レベル5で全種類）
* 障害物ごとの速度倍率（見た目の速度差のみに使用）を追加
* ホログラム広告演出（レベル5以上で出現）を追加

## v0.6 ニアミス・コンボ

* ニアミス判定（ジャンプ中に障害物と接近した際の成立条件）を追加
* コンボシステム（連続ニアミスでボーナス増加、一定時間ニアミスがないとリセット）を追加
* コンボ段階に応じた演出（色・発光の変化）と「NICE!」「COMBO ×N」ポップアップを追加
* スコアのマイルストーン演出（1000 / 3000 / 5000 / 10000到達時）を追加
* レベルアップ時のリング演出を追加

## v0.5 猫マスコット改善

* プレイヤーを猫らしいシルエット（耳・尾・肉球・ヒゲ）で再描画
* 表情（通常・まばたき・笑顔・ジャンプ中）を状況に応じて切り替え
* 走行アニメーション（肉球の位置切り替え）・尾の揺れアニメーションを追加
* 着地時のハッピー表情・着地パーティクルを追加

## v0.4 レベルシステム

* スコアに応じたレベル算出とレベルアップ演出（「LEVEL UP!」テキスト）を追加
* レベルに応じた障害物速度の上昇カーブを追加
* レベルに応じた背景変化（流れ星、ビル群、空の色合い変化）を追加
* ゲームオーバー時のフラッシュ演出を追加

## v0.3 背景演出

* ネオンカラーの星空・流れ星演出を追加
* 月・地平線・シンセウェイブ風パースグリッドを追加
* ジャンプ中の軌跡（トレイル）・ジャンプパーティクルを追加

## v0.2 ネオンテーマ

* ネオンカラーパレット（シアン・マゼンタ・パープル）と発光表現（shadowBlur）を導入
* Google Fonts「Orbitron」を導入しUIフォントを統一
* ハイスコアの `localStorage` 保存を追加
* 障害物のランダム生成・スコアに応じた速度上昇を追加

## v0.1 基本ゲームループ

* Canvas のセットアップ、地面・プレイヤーの描画を追加
* 重力・ジャンプ操作（スペースキー）を追加
* 障害物の移動と衝突判定を追加
* スコア加算と基本的なゲームループ（`update` → `render`）を追加
