# NEON NEKO RUNNER

ネオン×サイバーパンクな世界観で猫を操作する、ブラウザで遊べるシンプルな障害物回避アクションゲームです。

## ゲーム概要

* 自動で走り続ける猫を操作し、迫ってくる障害物をジャンプで回避してスコアを伸ばします。
* 道中に出現する魚を集めるとボーナススコアが入ります（レア魚はさらに高得点）。
* プレイを続けるとレベルが上がり、背景や障害物の種類が変化していきます。
* ニアミス・コンボ・実績解除・ミッション・ランキングなど、繰り返し遊びたくなる要素を搭載しています。
* ハイスコアや収集データ、統計はすべてブラウザの `localStorage` に保存され、次回プレイ以降も保持されます。

## 操作方法

| 操作 | 内容 |
| --- | --- |
| スペースキー / クリック / タップ | ジャンプ（空中で1回だけ追加ジャンプ可能） |
| スペースキー / クリック / タップ | タイトル・ゲームオーバー画面でスタート |
| P キー | 一時停止 / 再開 |
| M キー | タイトル画面からモード選択画面の表示 |
| K キー | タイトル画面からスキン選択画面の表示 |
| C キー | タイトル画面からミッション画面の表示 |
| S キー | タイトル画面から設定画面の表示 |
| A キー | タイトル画面から実績一覧の表示 |
| R キー | タイトル画面からランキング画面の表示 |
| T キー | タイトル画面から統計（Statistics）画面の表示 |
| ESC キー | 各画面からタイトル（または設定画面）へ戻る |
| ↑ / ↓ / ← / → キー | 各選択画面で項目・モード・スキンなどを切替 |

PC・モバイルの両方に対応しています。

## ゲームモード

ゲーム開始前に、タイトル画面から M キーでプレイスタイルを選べます。

| モード名 | 説明 | 特徴 |
| --- | --- | --- |
| Classic | Standard gameplay. | 標準のゲームバランス |
| Hard | Faster and more challenging. | 障害物速度+25%・魚/アイテム速度+15%・スコア1.5倍。レベルアップのペースはClassicと同じ |
| Time Attack | Score as much as possible before time runs out. | 60秒の時間制限。時間切れで「TIME UP!」として終了し、専用ランキングに記録される |

選択したモードは `localStorage` に保存され、次回起動時も引き継がれます。

## アイテム紹介

ゲーム中に一定確率でアイテムが出現します（画面内には常に最大1個）。

| アイテム名 | 効果時間 | 効果 |
| --- | --- | --- |
| Shield | 5秒 | 障害物に衝突してもゲームオーバーにならず、1回だけ防ぐ |
| Magnet | 8秒 | 周囲の魚を自動的に引き寄せて回収する |
| Score Boost | 10秒 | スコア獲得量（生存時間・ニアミス・魚収集）が2倍になる |
| Slow Time | 6秒 | 障害物・魚・アイテムの移動速度が約70%に低下する |

## スキン

プレイヤーの見た目（色）を切り替えられます。タイトル画面から K キーでスキン選択画面へ。

| スキン名 | 解放条件 |
| --- | --- |
| Cyber Cat | 初期解放 |
| Pink Cat | ハイスコア3000以上、またはミッション「Fish Collector」達成 |
| Golden Cat | 累計レア魚取得数10匹以上、またはミッション「Rare Hunter」達成 |
| Shadow Cat | 全実績解除、またはミッション「Survivor」達成 |

## 実績

5種類の実績を用意しています。タイトル画面から A キーで一覧を確認できます。

| タイトル | 解除条件 |
| --- | --- |
| First Jump | 初めてジャンプする |
| Combo Master | コンボ数10以上 |
| Fish Hunter | 累計魚取得数10以上 |
| Rare Collector | レア魚取得数1以上 |
| Cyber Legend | スコア10000以上 |

## ミッション

実績とは別に、プレイの目標となるミッションを用意しています。タイトル画面から C キーで一覧を確認できます。

| ミッション | 内容 | 報酬 |
| --- | --- | --- |
| Fish Collector | 魚を100匹集める（累計） | Pink Cat（スキン解放） |
| Hard Challenger | Hardモードで5000点達成 | +500 Bonus Score（表示のみ） |
| Rare Hunter | 累計レア魚20匹 | Golden Cat（スキン解放） |
| Survivor | Shieldで5回ゲームオーバーを防ぐ | Shadow Cat（スキン解放） |

## ランキング

Time Attackモードのスコアを上位5件まで記録する専用ランキングです。タイトル画面から R キーで確認できます。各エントリにはスコアと記録日が保存されます。

## 統計

これまでのプレイの積み重ねを振り返れる統計画面です。タイトル画面から T キーで確認できます。

* 総プレイ回数
* 総プレイ時間
* 総ジャンプ回数
* 総魚取得数
* 総レア魚取得数
* 最高スコア
* 最高レベル
* 最もプレイしたモード

## セーブデータのリセット

設定画面（S キー）から「RESET SAVE DATA」を選択すると確認画面が表示され、「YES」を選ぶと `localStorage` のすべてのデータ（ハイスコア・実績・スキン・ミッション・ランキング・統計など）を初期化してタイトル画面に戻ります。

## 主な機能

* ネオン×サイバーパンクな世界観（星空・ビル群・ホログラム広告など）
* レベルシステムによる背景・難易度の変化
* ニアミス判定とコンボボーナス
* 通常魚・レア魚の収集とスコアボーナス
* 障害物バリエーション（複数種類、レベルに応じて出現）

* ハイスコア・収集データ・統計のローカル保存（localStorage）

### GitHub Pagesでの公開方法

1. GitHub のリポジトリ画面で **Settings > Pages** を開く。
2. **Source** を `Deploy from a branch` に設定する。
3. ブランチを `main`、フォルダを `/ (root)` に設定して **Save** する。
4. しばらく待つと `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される。

ビルド手順は不要で、リポジトリの内容がそのまま公開されます。
